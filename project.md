# Проект: Цифровые инструменты риелтора

Личный маркетинговый проект Виктории Полтавченко — агента по недвижимости, работающего напрямую с застройщиком загородных домов в Подмосковье (поселок Тихие Луга). Все страницы созданы методом вайб-кодинга на HTML/CSS/JS без использования фреймворков. Проект размещён на GitHub: https://github.com/vikavka1970-lab/my-first-projekt

## Для кого

Для потенциальных покупателей загородных домов в Подмосковье, которые хотят купить дом напрямую от застройщика без ипотеки и переплат — в рассрочку на 36 месяцев.

---

## Страницы проекта

### Готовые страницы

| Файл | Назначение |
|------|-----------|
| `vizitka.html` | Цифровая визитка Виктории — имя, фото, контакты (Telegram, WhatsApp, телефон), краткое УТП, ссылка на лендинг |
| `landing.html` | Полноценный лендинг для продажи домов: галерея, акции, калькулятор рассрочки, видео с экскурсии, блок доверия, контакт |
| `voice-input.html` | Инструмент для голосового ввода текста (Web Speech API, ru-RU) — диктовка и копирование в буфер обмена |

### Вспомогательные скрипты

| Файл | Назначение |
|------|-----------|
| `start-server.ps1` | Локальный HTTP-сервер на порту 8080 — для просмотра сайта с телефона по Wi-Fi (192.168.88.178:8080) |
| `start-voice.ps1` | Локальный HTTP-сервер на порту 8765 — для запуска голосового ввода в Chromium |

### Планируемые страницы

- Публикация на GitHub Pages — чтобы лендинг и визитка были доступны по постоянной ссылке без локального сервера
- Страница/слайды для отправки клиентам в Telegram

## Стиль дизайна

**Лендинг (`landing.html`)**
- Тёмная природная тема: фон `#112318` (тёмно-зелёный), акценты — зелёный `#52b788` и золотой `#d4a017`
- Радиальные градиенты создают объёмное свечение на фоне
- Анимированный градиентный заголовок (shimmer-эффект)
- Карточки со стекло-морфизмом (glassmorphism): полупрозрачный фон + backdrop-filter blur
- Hover-эффекты: подъём карточки + свечение
- Адаптивная сетка (CSS Grid, auto-fit)
- Фиксированная навигация с blur-эффектом
- Плавающие кнопки Telegram/WhatsApp в правом нижнем углу

**Визитка (`vizitka.html`)**
- Тёмный фон `#0a0a0a`, анимированный градиентный фон
- Прямоугольное фото с эффектом рамки
- Градиентное имя (фиолетово-бирюзовый)
- Иконки соцсетей с hover-свечением
- Минималистичный стиль — всё на одном экране

---

## Контент

Все данные (цены, условия рассрочки, акции) строго взяты из официального каталога застройщика и сайта tlug.ru. Ничего не выдумано.

**Калькулятор рассрочки** — три типа домов (Шале, Барнхаус, Хай-тек), четыре площади (55, 75, 89, 110 м²), точные данные из PDF-каталога.

**Акция** — до 12 апреля: бассейн или участок земли на выбор при записи на экскурсию.
