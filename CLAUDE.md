# Правила работы с проектом

## Структура файлов
- Каждая страница — один самодостаточный HTML-файл (CSS и JS внутри, без внешних библиотек)
- Никаких фреймворков (React, Vue, Bootstrap и т.д.) — только чистый HTML/CSS/JS

## Дизайн
- Фон: тёмный
- Текст: светлый
- Акцентные цвета: зелёный `#52b788`, золотой `#d4a017` — на лендинге; фиолетово-бирюзовый — на визитке
- Адаптивная вёрстка: всё должно корректно отображаться на мобильных устройствах
- Используй CSS Grid или Flexbox с `auto-fit` / `minmax` для адаптивных сеток

## Текст на страницах
- Простой и дружеский язык — без канцелярита и сложных слов
- Пишем так, как говорят живые люди
- Только реальные данные из каталога застройщика и сайта tlug.ru — ничего не выдумывать

## Процесс работы
- Перед созданием нового файла — сначала описать план: что будет на странице, какие секции, какая логика
- После каждого значимого изменения — коммит и пуш на GitHub с понятным сообщением на русском языке
- Не добавлять функции, которые не просили

## Фотографии
- Все фото хранятся в папке `photos/`
- Фото домов: `1.png` — `10.png`
- Фото Виктории: `photos/victoria.png`
- Не встраивать фото через base64 — только через `src="photos/..."`
